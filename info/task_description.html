Help Peter to automate his task with creating module. As you could see on illustration, gray circle means on, and white circle means off. Every digit in time is converted to binary representation. Then, for ease of this task, you will converted every on ( or 1 ) signal to underscore ( _ ) and every off ( or 0 ) signal to dot ( . )

Input: A time string

Output: A string of Morse symbol from binary representation with specific format

Example: 

assert checkio( "10:37:49" ) == "._ .... : .__ .___ : _.. _.._", "First Test"
assert checkio( "21:34:56" ) == "-. ...- : .-- .-.. : -.- .--.", "Second Test"
assert checkio( "00:01:02" ) == ".. .... : ... ..._ : ... ..-.", "Third Test"
assert checkio( "23:59:59" ) == "-. ..-- : -.- -..- : -.- -..-", "Fourth Test"
